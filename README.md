Project Overview

This is a simple Virtual Reality (VR) project developed using Unity to demonstrate the basics of VR application development. The project focuses on creating an immersive 3D environment where users can explore and interact using a VR headset and controllers.

Technologies Used

Game Engine: Unity

Programming Language: C#

VR SDK: Unity XR / OpenXR

Platform: PC-based VR

Features

Immersive 3D virtual environment

Smooth player movement and navigation

Basic object interaction using VR controllers

Realistic lighting and environment setup

Comfortable user experience with optimized performance
